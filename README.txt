First you have to create file called .env and fill it with next parameters:

TOKEN=token can be obtained on Discord Developer portal, in your application
SERVER_ID=
CHANNEL_ID=text channel id where bot will handle slash-commands
EXE_PATH=path to .exe binary file for restart command
MUSIC_FOLDER=
SOUNDS_FOLDER=in progress
DOWNLOAD_FOLDER=in progress
PLAYLISTS_FOLDER=in progress
VOLUME=number from 0 to 100

If values has space symbols it should be quoted.
